                         TROUBLESHOOTING

Problem: Frontend can't connect to backend
┌─────────────────────────────────────────────────────────┐
│  ❌ Error: Network Error                                 │
│  ❌ CORS policy error                                    │
└─────────────────────────────────────────────────────────┘

Solution:
1. Check VITE_API_URL in Vercel (no trailing slash)
2. Check FRONTEND_URL in Render (no trailing slash)
3. Redeploy both services
4. Check browser console (F12) for details

────────────────────────────────────────────────────────────

Problem: Backend is slow
┌─────────────────────────────────────────────────────────┐
│  ⏳ Request taking 30-60 seconds                         │
└─────────────────────────────────────────────────────────┘

Solution:
✅ This is NORMAL for Render free tier (cold start)
✅ Subsequent requests will be faster
💡 Use UptimeRobot to keep backend warm

────────────────────────────────────────────────────────────

Problem: Deployment fails
┌─────────────────────────────────────────────────────────┐
│  ❌ Build failed                                         │
└─────────────────────────────────────────────────────────┘

Solution:
1. Check deployment logs in dashboard
2. Verify requirements.txt is correct
3. Retry deployment (PyTorch may timeout)
4. Clear build cache and redeploy
